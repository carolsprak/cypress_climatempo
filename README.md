# cypress_climatempo

### Acesse o tutorial de instalação para Ruby On Rails
https://app.gitbook.com/@carolsprak/s/ruby-on-rails-para-iniciantes/

#### Instale ruby version = 2.3
#### Instale  rails version = 5

### Crie uma API Rest em Ruby On Rails para visualizar o Clima e Tempo de uma cidade, veja o tutorial aqui:
https://medium.com/gtsw/como-testar-uma-rails-api-com-rspec-596d4ce39988


#### Ou faça o download ou clone do repositório climatempoapi.
git clone https://github.com/carolsprak/climatempoapi.git

#### Ou acesse diretamente a API: https://openweathermap.org/api

### Instale o Node.js (https://nodejs.org/)

#### Entre no diretório do cypress_climatempoapi e execute o comando no terminal:
> npm install

#### Quando o cypress terminar de instalar, execute o comando:
> npm run cypress:run

#### Se quiser ver o resultado dos testes no navegador, execute o comando:
> npm run cypress:open
